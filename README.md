# soldajustica #
Site oficial da igreja sol da justiça

## Requisitos ##
* Python 3.4

### Instalação ###
Instalação de dependências do projeto.  
```shell
$ pip install requirements.txt
```

Executando projeto no ambiente de desenvolvimento.  
```shell
$ python soldajustica/manage.py runserver
```
